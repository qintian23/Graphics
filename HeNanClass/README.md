# 计算机图形学

## 实验1  MFC框架的使用及实现金刚石算法（4学时）        验证性
实验内容：讲解MFC框架各个类的作用，在客户区上画出金刚石图案，在view类中实现其算法。 
基本要求： 
1. 掌握框架下自动生成的个各类的命名特征、特点、作用；
2. 了解各个类之间的函数调用顺序并了解怎么对客户区的进行修改；
3. 掌握金刚石算法的原理；
4. 通过交互式的输入，实现不同条件下的金刚石图案；
实验拓展：实现一个金刚石算法。


## 实验2  绘制任意斜率的直线（2学时）        验证性
实验内容：利用直线中点bresenham算法的思想，编程实现绘制一条直线。
基本要求：
1. 了解直线中点bresenham算法的思想；
2. 通过鼠标点击在客户区画一条直线；
3. 了解鼠标点击事件，及映射机制
实验拓展：圆和椭圆的中点bresenham算法。


## 实验3  交互式绘制多边形（2学时）        验证性
实验内容：利用边缘填充算法的思想，编程实现多边形的填充。
基本要求：
1. 了解边缘填充的思想；
2. 了解查找一个点是否在方形框里的判断方法；
实验拓展：四邻接点种子填充和八邻接点种子填充算法。


## 实验4  二维图形的几何变换  （2学时）       验证性
实验内容：通过工具栏上的按钮实现一个矩形的二维变换。
基本要求：
1. 掌握几何变换的矩阵乘法原理；
2. 在屏幕上通过工具栏按钮实现平移，错切，对称，放大等基本变换；
实验拓展：同时实现多个基本变换。


## 实验5  直线段裁剪  （2学时）       验证性
实验内容：使用Cohen_Sutherland裁剪算法裁剪直线段。
基本要求：
1. 掌握Cohen_Sutherland裁剪算法原理；
2. 在屏幕上固定绘制矩形代表窗口，使用鼠标绘制任意直线段（Bresenham算法实现），使用裁剪按钮裁剪直线段，去掉窗口外的部分；
实验拓展：在屏幕上显示一个直线构成的图案，使用鼠标选择两点绘制矩形代表窗口对图案矩形裁剪。
